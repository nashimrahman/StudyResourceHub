# Study Resource Sharing Hub 🎓

The **Study Resource Sharing Hub** is a student-driven platform designed to simplify access to academic resources while encouraging peer-to-peer contribution within a campus. The platform enables students to easily search, upload, and download materials such as notes, previous year question papers, books, and cheat sheets in a centralized and organized environment. 

With a clean, search-first interface and structured resource cards, users can quickly find relevant content using filters like department, semester, and resource type. The contribution flow is intentionally simple and frictionless, allowing students to upload resources through a lightweight form integrated with Google Sheets, ensuring scalability without complex infrastructure. 

By focusing on accessibility, minimal design, and fast navigation, the platform creates a seamless experience tailored for exam preparation and collaborative learning. Ultimately, it fosters a culture of sharing and collective growth, transforming scattered study materials into a unified, student-powered knowledge ecosystem.

## 🚀 Features

- **Search-First Library:** Quickly find exactly what you need with instant search and multi-tag filtering (Department, Year, Type).
- **Serverless Architecture:** Utilizes Google Sheets and Google Forms for a completely free, highly scalable, and frictionless backend.
- **Lightning Fast:** Implements local caching to ensure zero wait times when revisiting the application.
- **Local Bookmarks:** Save important notes for later with a persistent local bookmark system.
- **Simulated Upvotes:** Interactive upvote system built for peer-to-peer engagement.
- **Seamless Sharing:** One-click copy link to clipboard with animated toast notifications.

## 🛠️ Technology Stack

- **Frontend:** HTML5, JavaScript (Vanilla), Tailwind CSS
- **Backend/Database:** Google Sheets via SheetDB API
- **File Storage & Uploads:** Google Forms & Google Drive
- **State Management:** LocalStorage API

## 🏃‍♂️ Running Locally

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build steps or `npm install` required!
